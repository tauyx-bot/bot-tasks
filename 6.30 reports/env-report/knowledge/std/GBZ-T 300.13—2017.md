## 4 铟及其化合物的酸消解-火焰原子吸收光谱法

## 4.1 原理

空气中气溶胶态铟及其化合物用微孔滤膜采集，酸消解后, 用乙炔-空气火焰原子吸收分光光度计,在325.6 nm 波长下测定吸光度，进行定量。

## 4.2 仪器

4.2.1 微孔滤膜，孔径 0.8 µm。

4.2.2 大采样夹，滤料直径为 37 mm 或 40 mm。

4.2.3 小采样夹，滤料直径为 25 mm。

4.2.4 空气采样器，流量范围为 0 L/min～2 L/min 和 0 L/min～10 L/min。

4.2.5 烧杯，50 mL。

4.2.6 控温电热器。

4.2.7 具塞刻度试管，10 mL。

4.2.8 原子吸收分光光度计，具乙炔-空气火焰燃烧器和铟空心阴极灯。

## 4.3 试剂

4.3.1 实验用水为去离子水，用酸为优级纯。

4.3.2 硝酸， $\rho _ { \ 2 0 } { = } 1 . 4 2 \mathrm { \ g / m L }$ C

4.3.3 硝酸溶液，1%（体积分数）。

4.3.4 标准溶液：用硝酸溶液稀释国家认可的铟标准溶液成 $1 0 0 . 0 ~ \mu \mathrm { g / m I }$ 铟标准应用液。

## 4.4 样品的采集、运输与保存

4.4.1 现场采样按照 GBZ 159 执行。

4.4.2 短时间采样：在采样点，用装好微孔滤膜的大采样夹，以 5.0L/min 流量采集15min空气样品。

4.4.3 长时间采样：在采样点，用装好微孔滤膜的小采样夹，以1.0L/min 流量采集2h～8h空气样品。

4.4.4 采样后，打开采样夹，取出微孔滤膜，接尘面朝里对折两次，放入清洁的塑料袋或纸袋中，置清洁容器内运输和保存。样品在常温下可长期保存。

4.4.5 样品空白：在采样点，打开装好微孔滤膜的采样夹，立即取出滤膜，放入清洁的塑料袋或纸袋中，然后同样品一起运输、保存和测定。每批次样品不少于 2个样品空白。

## 4.5 分析步骤

4.5.1 样品处理：将采过样的微孔滤膜放入烧杯中, 加入5 mL硝酸, 盖上表面皿, 在控温电热器上150℃左右消解；待硝酸基本挥发干时, 立即取下。用硝酸溶液将残液定量转移至刻度试管中，并稀释至10.0 mL。样品溶液供测定。

4.5.2 标准曲线的制备：取 5支～8支10 mL 容量瓶，分别加入0.0 mL～2.0 mL铟标准应用液，用硝酸溶液定容，配制成 $0 . 0 \ \mu \mathrm { g / m L } { \sim } 2 0 . 0 \ \mu \mathrm { g / m L }$ 浓度范围的铟标准系列。将原子吸收分光光度计调节至最佳测定状态，在325.6nm波长下, 用乙炔-空气贫燃气火焰分别测定标准系列各浓度的吸光度。以测得的吸光度对相应的铟浓度 $( \mu \mathrm { g / m L } )$ 绘制标准曲线或计算回归方程，其相关系数应≥0.999。

4.5.3 样品测定：用测定标准系列的操作条件测定样品溶液和样品空白溶液，测得的吸光度值由标准曲线或回归方程得样品溶液中铟的浓度 ${ \mathrm { ( ~ } } \mu \mathrm { { g } / m L \mathrm { { ) } } }$ 。若样品溶液中铟浓度超过测定范围，用硝酸溶液稀释后测定，计算时乘以稀释倍数。

## 4.6 计算

4.6.1 按 GBZ 159的方法和要求将采样体积换算成标准采样体积。

4.6.2 按式（1）计算空气中铟的浓度：

$$
C = \frac {1 0 C _ {0}}{V _ {0}}\tag{1}
$$

式中：

C ——空气中铟的浓度，单位为毫克每立方米 $( \mathrm { m g / m ^ { 3 } } )$

10——样品溶液的体积，单位为毫升（mL）；

$C _ { O }$ \_ 测得的样品溶液中铟的浓度（减去样品空白）, 单位为微克每毫升 ${ \mathrm { ( ~ } } \mu \mathrm { { g } / m L \mathrm { { ) } } }$ ；

$V _ { 0 }$ ——标准采样体积，单位为升（L）。

4.6.3 空气中的时间加权平均接触浓度 $( C _ { \mathrm { T W A } } )$ 按 GBZ 159 规定计算。

## 4.7 说明

4.7.1 本法按照GBZ/T 210.4 的方法和要求进行研制。本法的检出限为 $0 . 4 ~ \mu \mathrm { g } / \mathrm { m l }$ ，定量下限为 1.3$\mu \mathrm { g } / \mathrm { m L }$ ，定量测定范围为 $1 . 3 ~ { \mu \mathrm { g / m L } } { \sim } 2 0 ~ { \mu \mathrm { g / m L } }$ ；以采集75L空气样品计，最低检出浓度为 $0 . 0 5 3 \mathrm { m g / m ^ { 3 } }$ 最低定量浓度为 $0 . 1 8 ~ \mathrm { \ m g / m ^ { 3 } }$ ；相对标准偏差为 $1 . 0 \% \sim 7 . 7 \%$ ，采样效率为 99.9%～100％。

4.7.2 样品也可采用微波消解方法。

4.7.3 在样品溶液中，铟浓度为 $4 . 5 ~ \mu \mathrm { g } / \mathrm { m l }$ 时，小于 $2 2 5 \mathrm { \mu g / m L M g ^ { 2 + } }$ $6 0 ~ \mathrm { \mu g / m L C u } ^ { 2 + }$ $\mathrm { A l } ^ { 3 + }$ $Z \mathrm { n } ^ { 2 + }$ 和 15$\mu \mathrm { g } / \mathrm { m l }$ 磷酸不干扰测定。
