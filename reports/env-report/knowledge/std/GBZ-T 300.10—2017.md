## 4 钴及其化合物的酸消解-火焰原子吸收光谱法

## 4.1 原理

空气中气溶胶态钴及其化合物用微孔滤膜采集，消解后，用乙炔-空气火焰原子吸收分光光度计,在240.7 nm波长下测定吸光度，进行定量。

## 4.2 仪器

4.2.1 微孔滤膜，孔径 0.8 μm。

4.2.2 大采样夹，滤料直径为 37 mm 或 40 mm。

4.2.3 小采样夹，滤料直径为25 mm。

4.2.4 空气采样器，流量范围 0 L/min～2 L/min 和 0 $_ { \mathrm { L } / \mathrm { m i n } \sim 1 0 ~ \mathrm { L } / \mathrm { m i n } }$ 。

4.2.5 烧杯，50 mL。

4.2.6 控温电热器。

4.2.7 具塞刻度试管，25 mL。

4.2.8 原子吸收分光光度计，具乙炔-空气火焰燃烧器和钴空心阴极灯。

## 4.3 试剂

4.3.1 实验用水为去离子水，用酸为优级纯，试剂为分析纯。

4.3.2 消解液：1体积高氯酸 $( \mathrm { ~ \ } \rho _ { \ 2 0 } { = } 1 . 6 7 \mathrm { ~ \ g / m L } )$ 与9体积硝酸 $( \ \rho _ { \ 2 0 } { = } 1 . 4 2 \ \mathrm { \ g / m L } )$ 混合。

4.3.3 盐酸羟胺溶液， $2 0 0 \mathrm { ~ g / L ~ }$ C

4.3.4 硝酸溶液，0.48 mol/L。

4.3.5 标准溶液：用硝酸溶液稀释国家认可的钴标准溶液成 $5 0 . 0 ~ \mu \mathrm { g } / \mathrm { m l }$ 钴标准应用液。

## 4.4 样品的采集、运输和保存

4.4.1 现场采样按照 GBZ 159 执行。

4.4.2 短时间采样：在采样点，用装好微孔滤膜的大采样夹，以5.0L/min流量采集15 min空气样品。

4.4.3 长时间采样：在采样点，用装好微孔滤膜的小采样夹，以 $1 . 0 ~ \mathrm { L / m i n }$ 流量采集 $2 ~ \mathrm { h } { \sim } 8 ~ \mathrm { h }$ 空气样品。

4.4.4 采样后，打开采样夹，取出微孔滤膜，接尘面朝里对折两次，放入清洁的塑料袋或纸袋中，置清洁容器内运输和保存。样品在室温下可长期保存。

4.4.5 样品空白：在采样点，打开装好微孔滤膜的采样夹，立即取出滤膜，放入清洁的塑料袋或纸袋中，然后同样品一起运输、保存和测定。每批次样品不少于2个样品空白。

## 4.5 分析步骤

4.5.1 样品处理：将采过样的微孔滤膜放入烧杯中，加入5mL消解液，盖好表面皿，在控温电热器上200C左右消解，待消解液基本挥发干时，取下稍冷后，加 2滴盐酸羟胺溶液和少量硝酸溶液；加热溶解残液，用硝酸溶液定量转移入具塞刻度试管中，并稀释至 25.0 mL。样品溶液供测定。

4.5.2 标准曲线的制备：取5支～8支25 mL 容量瓶，分别加入 $0 . 0 \ \mathrm { m L } { \sim } 5 . 0 \ \mathrm { m L }$ 钴标准应用液，各加硝酸溶液至刻度，配成 $0 . 0 ~ \mu \mathrm { g / m L } { \sim } 1 0 . 0 ~ \mu \mathrm { g / m L }$ L浓度范围的钴标准系列。将原子吸收分光光度计调节至最佳测定状态，在 240.7nm波长下，用乙炔-空气火焰分别测定标准系列各浓度的吸光度。以测得的吸光度对相应的钴浓度 $\mathrm { { ( \mu g / m L ) } }$ 绘制标准曲线或计算回归方程，其相关系数应≥0.999。

4.5.3 样品测定：用测定标准系列的操作条件测定样品溶液和样品空白溶液，测得的吸光度值由标准曲线或回归方程得样品溶液中钴的浓度 $( \mu \mathrm { g } / \mu \mathrm { L } )$ 。若样品溶液中钴浓度超过测定范围，用硝酸溶液稀释后测定，计算时乘以稀释倍数。

## 4.6 计算

4.6.1 按 GBZ 159的方法和要求将采样体积换算成标准采样体积。

4.6.2 按式（1）计算空气中钴的浓度：

$$
C = \frac {2 5 C _ {0}}{V _ {0}}\tag{1}
$$

式中：

——空气中钴的浓度，单位为毫克每立方米 $\mathrm { { ( m g / m ^ { 3 } ) } }$

25——样品溶液的体积，单位为毫升（mL）；

$C _ { \sigma }$ ——测得的样品溶液中钴的浓度（减去样品空白），单位为微克每毫升 $( \mu \mathrm { g } / \mathrm { m L } )$ ；

$V _ { 0 ^ { - } }$ ——标准采样体积，单位为升（L）。

4.6.3 空气中的时间加权平均接触浓度 $( \mathit { C } _ { \mathrm { I W A } } )$ 按 GBZ 159 规定计算。

## 4.7 说明

4.7.1 本法按照 GBZ/T 210.4 的方法和要求进行研制。本法的检出限为 0.02 μg/mL，定量下限为 0.07$\mu \mathrm { g } / \mathrm { m L }$ ，定量测定范围为 $0 . 0 7 \ \mu \mathrm { g / m L } { \sim } 1 0 . 0 \ \mu \mathrm { g / m L }$ ；以采集75 L空气样品计，最低检出浓度为 0.007$\mathrm { m g / m ^ { 3 } }$ ，最低定量浓度为 $0 . 0 2 ~ \mathrm { m g / m ^ { 3 } }$ ；平均相对标准偏差为 4.4％，平均采样效率≥99％。

4.7.2 样品溶液中 $2 . 5 \mathrm { \ m g / m L \ S i } ^ { 4 + }$ ，2 mg/mL Co<sup>2+</sup>、Mo<sup>2+</sup>、V<sup>5+</sup>，0.6 mg/mL Ni<sup>2+</sup>，0.5 mg/mL Cu<sup>2+</sup>、Mn<sup>2+</sup>，$0 . \mathrm { { 1 } \ m g / m L \ C a ^ { 2 + } }$ 不干扰测定。
