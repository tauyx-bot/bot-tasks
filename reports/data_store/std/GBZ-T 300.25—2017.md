## 4 铊及其化合物的溶剂洗脱-石墨炉原子吸收光谱法

## 4.1 原理

空气中气溶胶态铊及其化合物用微孔滤膜采集，酸洗脱后，用石墨炉原子吸收分光光度计在276.7nm 波长下测定吸光度，进行定量。

## 4.2 仪器

4.2.1 微孔滤膜，孔径 0.8 μm。

4.2.2 大采样夹，滤料直径为 37 mm 或 40 mm。

4.2.3 小采样夹，滤料直径为25 mm。

4.2.4 空气采样器，流量范围为 0 L/min～2 L/min 和 0 L/min～10 L/min。

4.2.5 具塞刻度试管，5 mL。

4.2.6 原子吸收分光光度计，具石墨炉原子化器、背景校正装置和铊空心阴极灯，仪器操作参考条件：

a) 干燥： $8 0 ^ { \circ } \mathrm { C } \sim 1 2 0 ^ { \circ } \mathrm { C } , 2 0 \mathrm { s } ;$ 保持 10 s；

b) 灰化： $1 2 0 ^ { \circ } \mathrm { C } \sim 3 0 0 ^ { \circ } \mathrm { C } , 2 0 { \mathrm { s } } ;$ 保持 10 s；

c) 原子化：1800℃，5s，停气；

d) 清除：2400℃，2 s。

## 4.3 试剂

4.3.1 实验用水为去离子水，用酸为优级纯。

4.3.2 硝酸溶液，0.5%(体积分数）。

4.3.3 标准溶液：用水稀释国家认可的铊标准溶液成 $1 . 0 ~ \mu \mathrm { g } / \mathrm { m L }$ 铊标准应用液。

## 4.4 样品的采集、运输和保存

4.4.1 现场采样按照 GBZ 159 执行。

4.4.2 短时间采样：在采样点，用装好微孔滤膜的大采样夹，以5.0L/min流量采集 15 min空气样品。

4.4.3 长时间采样：在采样点，用装好微孔滤膜的小采样夹，以 1.0 L/min流量采集 2 h～8 h空气样品。

4.4.4 采样后，打开采样夹，取出微孔滤膜，接尘面朝里对折，放入具塞刻度试管中，置清洁容器内运输和保存。样品在室温下可保存 15d。

4.4.5 样品空白：在采样点，打开装好微孔滤膜的采样夹，立即取出滤膜，放入具塞刻度试管中，然后同样品一起运输、保存和测定。每批次样品不少于 2 个样品空白。

## 4.5 分析步骤

4.5.1 样品处理：向装有微孔滤膜的具塞刻度试管中加入5.0 mL硝酸溶液，强烈振摇，放置 5 min。样品溶液供测定。

4.5.2 标准曲线的制备：取5支～8支5mL容量瓶，分别加入0.0 mL～0.50 mL铊标准应用液，用硫酸溶液定容，配成 $0 . 0 ~ \mu \mathrm { g / m L } { \sim } 0 . 1 0 ~ \mu \mathrm { g / m L }$ L浓度范围的铊标准系列。参照仪器操作条件，将石墨炉原子吸收分光光度计调节至最佳测定状态，使用背景校正，进样 $2 0 . 0 ~ \mu \mathrm { L }$ ，在 276.7 nm波长下，分别测定标准系列各浓度的吸光度。以测得的吸光度对相应的铊浓度 $( \mu \mathrm { g } / \mu \mathrm { L } )$ 绘制标准曲线或计算回归方程，其相关系数应≥0.99。

4.5.3 样品测定：用测定标准系列的操作条件测定样品溶液和样品空白溶液，测得的吸光度值由标准曲线或回归方程得样品溶液中铊的浓度 ${ \mathrm { ( } } \mu \mathrm { { g } / \mathrm { { m L } } } { \mathrm { ) } }$ 。若样品溶液中铊浓度超过测定范围，用硝酸溶液稀释后测定，计算时乘以稀释倍数。

## 4.6 计算

4.6.1 按 GBZ 159的方法和要求将采样体积换算成标准采样体积。

4.6.2 按式（1）计算空气中铊的浓度：

$$
C = \frac {5 C _ {0}}{V _ {0}}\tag{1}
$$

式中：

C——空气中铊的浓度，单位为毫克每立方米 $\mathrm { ( m g / m ^ { 3 } ) }$

5——样品溶液的体积，单位为毫升（mL）；

$C _ { \cal { O } }$ 测得的样品溶液中铊的浓度（减去样品空白），单位为微克每毫升 $( \mu \mathrm { g / m L } )$ ）；

$V _ { 0 }$ ——标准采样体积，单位为升（L）。

4.6.3 空气中的时间加权平均接触浓度（ ）按GBZ 159规定计算。

## 4.7 说明

4.7.1 本法按照 GBZ/T 210.4 的方法和要求进行研制。本法的检出限为 0.01 μg/mL，定量下限为 0.033$\mu \mathrm { g } / \mathrm { m L }$ ，定量测定范围为 $0 . 0 3 3 ~ \mu \mathrm { g / m L } { \sim } 0 . 1 0 ~ \mu \mathrm { g / m L }$ ；以采集75 L空气样品计，最低检出浓度为 0.0007$\mathrm { m g / m } ^ { 3 }$ ，最低定量浓度为 $0 . 0 0 2 ~ \mathrm { m g / m } ^ { 3 }$ ；相对标准偏差＜10%，采样效率≥98%。

4.7.2 因铊为易挥发元素，操作中不能使用盐酸或高氯酸，灰化温度不能超过 300℃。

4.7.3 本法适用于检测能够溶于硝酸溶液（4.3.2）的铊化合物。

4.7.4 铊盐毒性大，操作时应注意防护。
